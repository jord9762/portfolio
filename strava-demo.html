Strava Premium Tool

Project Overview

The Strava Premium Tool is a custom fitness analytics platform I developed to replicate and enhance Strava Premium features. By integrating running data visualization with geospatial mapping, this application provides insights that aren't available in standard fitness tracking platforms.

Key Features:

  • Interactive Data Visualizations: Plotly-powered charts including pie charts, line graphs, and bar charts to track performance metrics over time.
  • Custom Mapping: Folium-based maps with Public Rights of Way and forest overlays to analyze running routes in relation to terrain.
  • Advanced Route Analysis: GPX file parsing with speed-based coloring and elevation DEM overlays to identify performance patterns across different terrains.
  • Multi-technology Stack: Built with Python, JavaScript, HTML/CSS, and PostgreSQL for a robust and responsive user experience.
  • Personal Data Integration: Securely connects with the Strava API to import and analyze personal running data.

Technical Implementation Details

Activity Mix Analytics

A breakdown of exercise types to help guide training and highlight neglected areas. This visualization helps users maintain a balanced training regimen across different activity types.

Custom Effort Score System

A proprietary scoring system I designed that quantifies workout intensity based on heart rate data and duration, providing a more comprehensive measure of exercise effort than distance or pace alone.

PostGIS Database Schema

The underlying PostgreSQL/PostGIS database structure that powers the application, organizing all Strava metrics in a normalized schema for efficient querying and spatial analysis.

Interactive Map Loading System

Custom JavaScript that renders a loading bar while the Folium map initializes, improving user experience by providing visual feedback during complex geospatial data processing.

Robust SQL Data Retrieval

Optimized SQL queries that fetch different metrics from the database while handling null values gracefully to prevent visualization errors. This ensures reliable data for the application's charts and graphs.

Brand-Aligned UI Design

Carefully crafted CSS styling using an orange and black color scheme that aligns with Strava's brand identity while maintaining a unique and professional appearance for the application.
