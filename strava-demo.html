Strava Premium Tool

Application Tour

The Strava Premium Tool is a custom fitness analytics platform I developed to replicate and enhance Strava Premium features. I tried to replicate all the benefits of Strava premium using Strava's API but went beyond the standard feature set by adding features such as effort score by week and activity mix.

Key Features:

  • Interactive Data Visualizations: Plotly-powered charts including pie charts, line graphs, and bar charts to track performance metrics over time. All historic data is included allowing for in depth analysis.
  • Custom Mapping: Folium-based maps with Public Rights of Way and forest overlays to ive me ideas for new running routes.
  • Multi-technology Stack: Built with Python, JavaScript, HTML/CSS, and PostgreSQL for a robust and responsive user experience.
  • Personal Data Integration: Securely connects with the Strava API to import and analyze personal running data. I also recreated Strava's idea for an effort score by creating a composite index comprising of heart rate, time spent doing exercise and distance.

This project showcases my ability to combine my passion for running with technical skills in data visualization, database management, and web development. Was an extremely fun project and I hope to refine it further as I develop my software skills.
