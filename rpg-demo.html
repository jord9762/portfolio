Retro 90's Game

Application Tour

The Retro 90's Game is a passion project that combines my love for classic games with modern programming techniques. Built using Pygame, this 2D RPG features complex game mechanics including enemy AI, animation systems, and database-driven elements such as health, mp and damage.

Key Features:

  • Game Engine Development: Custom-built animation, collision detection/hitbox, and game state management systems.
  • Database Integration: SQLite-backed inventory systems, character stats tracking, and dialogue trees.
  • Strava Integration: Unique feature that links in-game character stats to real-world fitness achievements from Strava data (work in progress but successfully implemented basic functionality).
  • Custom Asset Design: Hand-crafted pixel art, animations, and sound effects for an authentic retro feel.
  • Development Documentation: Regular devlogs and gameplay footage to track the evolution of the game over time.

This project demonstrates not only my programming skills but also my ability to manage a complex technical project from concept through execution, incorporating database design, UI/UX considerations, and game mechanics into a cohesive product. Despite being a bit removed from GIS this project is probably the hardest one I have ever worked on due to the sheer complexity. I feel it has improved my ability to plan, research and problem-solve effectively. It has also greatly increased my knowledge surrounding both Python and SQL.
